# what css

1. css allows to create good looking web pages.
2. 3 ways to insert css
    1. External CSS
    2. Internal CSS
    3. Inline CSS
3. all `<p>` tags to be red text example
    p {
        color: red;
    }

## Example

Set the text-color for different elements:

body {
  color: red;
}

h1 {
  color: #00ff00;
}

p.ex {
  color: rgb(0,0,255);
}
